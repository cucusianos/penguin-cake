# v4 调整说明

本版按三张设计图重新打磨前台、中台、后台，并补齐下单链路。

## 前台

- 继续摈弃浅色甜品风，统一为 ORYZO 深色软木质感。
- 首页商品分类按钮已生效：全部蛋糕、巧克力、芝士、水果、慕斯。
- 商品卡片点击加入购物车，不再直接跳支付。
- 顶部购物车显示商品数量。

## 购物车与支付链路

- 新增 `/cart` 购物车页面。
- 支持增减数量、移除商品。
- 必填收货人、手机号、城市、详细地址、期望送达时间。
- 未填写配送信息时，`/checkout` 会阻止支付并提示返回购物车。
- 填写完整后进入统一收银台，支持微信、支付宝、Stripe、虚拟币、Mock 支付。

## 中台

- 按设计图补齐：珠三角配送全局地图、平台聚合、支付网关、验券中心。
- 补齐：蛋糕解耦中间件事件总线、事件流、订单状态看板。
- 补齐底部栏目：库存/预警、配送热力地图、服务健康状态、今日待办。

## 后台

- 按设计图补齐：销售额、销量、门店数、毛利率、货损率、配送准时率。
- 补齐：销售金额趋势、热销蛋糕排行、各门店销售对比、支付方式占比。
- 补齐：珠三角 12 城门店分布、蛋糕状态、原材料库存、配送管理。
- 补齐：门店销售概览、近期货损记录、今日进货记录。
