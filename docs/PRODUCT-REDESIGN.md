# 企鹅蛋糕 v3 产品打磨说明

## 视觉原则

1. 摈弃马卡龙浅色和卡通甜品感，改为真实蛋糕摄影。
2. 采用 ORYZO 风格：深色软木纹理、暖色文字、锈橙强调、低饱和橄榄绿状态色。
3. 组件统一：12px 卡片、36px 胶囊按钮、18px 元素间距、高密度但清晰的信息层级。
4. 蛋糕内容强调真实、质感、光影、奶油纹理、水果和巧克力细节。

## 三端拆分

- 前台：品牌展示、产品销售、订单状态。
- 中台：订单、支付、验券、外卖、配送、事件总线编排。
- 后台：报表、门店、仓库、货损、进货、原材料、配送管理。

## 页面路由

```text
/             前台官网
/login        登录与三端入口
/middle       中台运营首页
/back         后台管理首页
/order-status 订单状态
/checkout     支付演示
```
